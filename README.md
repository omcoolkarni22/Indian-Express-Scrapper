# Indian-Express-Scrapper

**Indian Express** is one among the Top-most News websites.</br>
This repository contains code that scrapes Indian Express News. Each topic page (Technology, Lifestyle, Cities, etc) is visited one after the other and the latest News on the first page is scarpped.</br>
All scrapped is stored in CSV file(storage can be changed as per need).</br>

**BeautifulSoup** a.k.a **BS4** and **Requests** are the major Modules used.</br>

# Execute it on your system

1. Clone or Download the Repository.
2. Install the requirements.txt file using CMD via __pip install -r requirements.txt__.
3. Now you're ready to Execute and Scrape news data from the website.
4. Final step, execute __python main.py__ on your system and wait for the results.
5. You'll have CSV files generated after a successfull scrape.
